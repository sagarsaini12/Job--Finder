# 🚀 Job Portal System (Django Full-Stack Project)

A scalable and feature-rich job portal application built using Django. This platform connects job seekers with recruiters and provides a complete hiring workflow including job posting, application tracking, and user management.

---

## 📌 Project Overview

This project is an open-source job portal system that supports multiple user roles with role-based access control. It simulates a real-world recruitment system with authentication, job management, and analytics features.

---

## ✨ Key Features

### 👤 Job Seeker Module
- Advanced job search with filters (location, skills, experience)
- Apply for jobs and track application status
- Create and manage professional profile
- Receive job alerts (email notifications)

### 🏢 Recruiter Module
- Post, update, and delete job listings
- Manage candidate applications
- Schedule interviews
- View analytics and job performance

### 🔐 Authentication & Security
- Secure user registration and login
- Email verification system
- Password reset functionality
- Role-Based Access Control (RBAC)

### ⚙️ System Features
- RESTful API architecture
- Background task processing using Celery
- Fast search powered by Elasticsearch
- Scalable and modular Django structure

---

## 🛠️ Tech Stack

### Backend
- Django (Python)
- Django REST Framework

### Database
- SQLite (Development)
- PostgreSQL (Production Ready)

### Tools & Technologies
- Redis (Caching & Task Queue)
- Celery (Background Jobs)
